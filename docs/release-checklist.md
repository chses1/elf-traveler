# 發布前檢查表

上傳 GitHub 或啟用 GitHub Pages 前，請用本機預覽網址完整檢查一次。

## 基本載入

- [ ] 首頁可以正常打開
- [ ] 登入背景圖正常顯示
- [ ] 首頁、地圖、戰鬥、勝利、失敗圖片沒有破圖
- [ ] 瀏覽器開發者工具沒有明顯紅色錯誤

## 主要流程

- [ ] 可以輸入 5 位數字學號登入
- [ ] 可以選擇男主角或女主角
- [ ] 可以進入桃園市 13 區地圖
- [ ] 可以點開至少一個行政區介紹
- [ ] 可以進入隊伍選擇畫面
- [ ] 可以開始一場戰鬥
- [ ] 可以回答題目並看到答題結果與解說
- [ ] 可以累積能量並使用技能
- [ ] 戰鬥勝利後可以收服精靈並獲得獎勵
- [ ] 圖鑑可以顯示已收服精靈
- [ ] 神社可以顯示角色與升級資訊

## 內容完整性

- [ ] 13 個行政區都出現在地圖或流程中
- [ ] 每區都有一隻守護精靈
- [ ] 每區至少有 5 題四選一題目
- [ ] 完成 13 區後，最終 Boss 入口可以出現

## 裝置檢查

- [ ] 桌機或筆電寬度下畫面正常
- [ ] 平板寬度下按鈕與文字沒有重疊
- [ ] 手機寬度下首頁、地圖按鈕與戰鬥畫面仍可操作

## GitHub Pages 上線後

- [ ] 公開網址可以打開遊戲首頁
- [ ] 圖片與樣式都有載入
- [ ] 登入、地圖、戰鬥、圖鑑、神社流程可用
- [ ] 重新整理頁面後，同一瀏覽器的進度仍保留
